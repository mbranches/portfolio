Meus Projetos

Departamento de Reparos API

API desenvolvida para a gestão de um Departamento de Reparos. A aplicação conta com relacionamentos complexos e diversas funcionalidades, tais como: controle de estoque de peça, conforme a mesma é utilizada em reparos, CRUD de Cliente, Funcionário, Veículo, Reparo e Peça. Permite, também, associar ou desassociar funcionários e peças a reparos específicos, além de realizar consultas avançadas, como consultar reparos por data, cliente e funcionário por nome.

Orçamentos SINAPI

Aplicativo desktop intuitivo desenvolvido para facilitar a criação de orçamentos de insumos voltados à construção civil. Com uma interface amigável e de fácil navegação, o sistema permite ao usuário gerar orçamentos detalhados de forma rápida e organizada. Entre suas principais funcionalidades, destacam-se: criação e edição de orçamentos, cadastro e gerenciamento de insumos (materiais e serviços), gerenciamento de clientes, podendo associa-los à orçamentos, editar seus dados e setar o tipo de cliente. Além do cálculo automático dos valores totais.

Ledger API

API desenvolvida como desafio da comunidade DevsNorte. A mesma conta com funcionalidades como cadastro de transações, listar histórico de transações, podendo filtrar por data, entrada ou saída. Conta, ainda, com a possibilidade de consulta de saldo, que faz uma soma de todas as transações, típico de um Ledger, e exportação de transações para Json ou CSV.
