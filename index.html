Departamento de Reparos API

API desenvolvida para a gestão de um Departamento de Reparos. A aplicação conta com relacionamentos complexos. A mesma tem diversas funcionaldades, tais como: controle de estoque, conforme a mesma é utilizada em reparos, CRUD de Cliente, Funcionário, Veículo, Reparo e Peça. Permite, também, associar ou desassociar funcionários e peças a reparos específicos, fazer consultar avançadas de reparos por data, cliente e funcionário por data.

Ledger API

API desenvolvida como desafio da comunidade DevsNorte. A mesma conta com funcionalidades como cadastro de transações, listar histórico de transações, podendo filtrar por data, entrada ou saída. Conta, ainda, com a possibilidade de consulta de saldo, que faz uma soma de todas as transações, típico de um Ledger, e exportação de transações para Json ou CSV.
